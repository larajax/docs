# Framework API

## Method - `ajax` {#ajax}

Issues a new AJAX request.

```js
jax.ajax('onSubmitContactForm')
```

## Method - `request` {#request}

Submit an AJAX request using a form.

```js
jax.request('#myForm', 'onSubmitContactForm')
```
